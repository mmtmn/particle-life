# Particle Life Presentation PDFs

These PDFs are dark themed presentation decks for audiences that have never
seen particle life before. They focus on the idea, the visual behavior, and the
high-level GPU simulation story.

Generated files:

- `particle-life-intro-deck.pdf`
- `emergence-visual-story.pdf`
- `cuda-simulation-story.pdf`

Regenerate them with:

```bash
python3 tools/make_pdfs.py
```

The PDF generator is dependency-free and draws the decks as vector PDF pages.
